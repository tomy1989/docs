## API Versioning

Our API uses semantic versioning. The current version is `v1` and all endpoints are prefixed with `/api/v1/`. We maintain backward compatibility and provide advance notice of any breaking changes.

## Rate Limits

To ensure optimal performance for all users, our API implements intelligent rate limiting:

- **Standard Plan**: 1,000 requests per minute
- **Professional Plan**: 5,000 requests per minute  
- **Enterprise Plan**: Custom limits based on your needs

Rate limit headers are included in all API responses to help you monitor your usage.
